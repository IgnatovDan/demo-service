Service for Xenarius demo applications

This demo service provides basic data for viewing and editing incident tickets by IT-system administrators.

Initialization

To generate a test database, run the following commands in Package Manager Console
Tools → Library Package Manager → Package Manager Console:

  • sqllocaldb.exe stop v11.0
  • sqllocaldb.exe delete v11.0
  • Update-database

Service API

The service provides data about incident tickets and persons. If demo applications don't work properly, please, ensure that these links work.

Demo applications

We provide two demo applications working with this service:

  1. The first application is the result of completing the Getting Started guide.
  2. The sesond application demonstrates a more complicated scenario described at Help Desk Demo.
